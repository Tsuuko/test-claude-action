# test-claude-action

こんにちは！このリポジトリへようこそ。

## 概要

このリポジトリは以下の目的があります：

- ClaudeCodeActionまわりの実行テスト
- 日本語コードレビュー機能のテスト

## 日本語コードレビュー機能

このリポジトリでは、Claude Codeが日本語でコードレビューを提供します。

### 使用方法

1. プルリクエストを作成
2. コメントで `@claude` をメンション
3. 日本語でのコードレビューを受け取る

### レビュー内容

- コード品質とベストプラクティス
- 潜在的なバグや問題の指摘
- パフォーマンスの改善案
- セキュリティの懸念事項
- テストカバレッジの確認

### 例

```python
# サンプルコードをレビューしてもらう場合
def calculate_total(items):
    total = 0
    for item in items:
        total += item.price
    return total
```

`@claude` このコードをレビューしてください。

→ 日本語でのレビューコメントが返されます
